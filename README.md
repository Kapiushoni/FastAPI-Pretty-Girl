# FastAPI Pretty Girl ✨

A simple FastAPI web application that displays a compliment for a "pretty girl".

## 🚀 Live Demo
You can view the live site here: **[https://fastapi-pretty-girl.onrender.com](https://fastapi-pretty-girl.onrender.com)**

> **Note on Deployment:** The site is hosted on a **Free Tier** (Render.com). The server goes into "sleep" mode after inactivity, so the **first load may take up to 50 seconds** while it wakes up.

## 🛠 Libraries Used
* **FastAPI** — Web framework
* **Uvicorn** — ASGI server
* **Jinja2** — HTML templating

## 📖 How to run locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
2. Run the server:
    ```bash
   uvicorn main:app --reload
